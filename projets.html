Systèmes Embarqués & IoT

Station Météo Connectée

Station de surveillance environnementale temps réel utilisant une architecture bi-processeur pour la performance.

  • Acquisition : Microcontrôleur STM32 pour la lecture précise des capteurs (Température, Humidité, Pression).
  • Communication : Transmission des données au module ESP32 via liaison UART.
  • Cloud : Envoi des données vers un Dashboard Web pour visualisation en direct via Wi-Fi.
STM32 (C/C++) ESP32 UART / I2C Web Dashboard
