Smart Water Management (AI + IoT)

Projet TAWFIR

3ème Prix ENSA El Jadida

Solution intelligente pour la gestion de la crise hydrique. Le système détecte les fuites domestiques en temps réel en analysant les habitudes de consommation.

  • IA Détective : Algorithme Isolation Forest pour isoler les anomalies de débit.
  • Architecture IoT : Capteur de débit + Arduino (Acquisition) + ESP32 (Transmission MQTT).
  • Backend : API Python/Flask et base de données MongoDB pour l'historique.
  • Contexte : Analyse vectorielle sur 60 minutes pour une précision maximale.
Machine Learning ESP32 / MQTT Python Flask MongoDB

Systèmes Embarqués & IoT

Station Météo IoT

Station de surveillance environnementale haute performance basée sur une architecture bi-cœur pour optimiser l'acquisition et la transmission.

  • Acquisition : Microcontrôleur STM32 pour la lecture précise des capteurs (Temp/Hum/Pression).
  • Communication : Liaison UART optimisée vers le module de communication.
  • Connectivité : Module ESP32 pour l'envoi sécurisé des données vers le Cloud (Dashboard Web).
STM32 (C/C++) ESP32 (Wi-Fi) UART / I2C IoT Dashboard
